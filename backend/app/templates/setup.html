⚙️ Setup Guide

Configure your API keys to start translating

1 Find the app folder

Locate the folder where jpeigo-slides.exe is running (or the unzipped folder if you're using the portable version). Inside you'll find:

📁 jpeigo-slides/
 ┣ 📄 jpeigo-slides.exe
 ┣ 📁 backend/
 ┃ ┣ 📄 .env.example  ← start here
 ┃ ┣ 📄 .env          ← create this
 ┃ ┗ …
 ┗ …

2 Create your .env file

Copy .env.example to .env and open it in a text editor (Notepad works fine).

cp backend\.env.example backend\.env

💡 On Windows: right-click in the folder → "Open in Terminal" → paste the command above.

3 Add an API key

You need at least one translation provider. Here's how to get a free Gemini key:

  1. Go to aistudio.google.com/apikey
  2. Sign in with your Google account
  3. Click "Create API Key"
  4. Copy the key and paste it into your .env file:
# backend/.env
GEMINI_API_KEY=AIzaSy...

Save the file. That's it!

4 Restart the app

Close the app window and run jpeigo-slides.exe again. The health indicator in the top-right corner will turn green when a key is detected.

⚙️ セットアップガイド

翻訳を開始するにはAPIキーを設定してください

1 アプリフォルダを開く

jpeigo-slides.exeがあるフォルダ（またはポータブル版を解凍したフォルダ）を開きます。以下の構成になっています：

📁 jpeigo-slides/
 ┣ 📄 jpeigo-slides.exe
 ┣ 📁 backend/
 ┃ ┣ 📄 .env.example  ← ここから始める
 ┃ ┣ 📄 .env          ← これを作成する
 ┃ ┗ …
 ┗ …

2 .env ファイルを作成

.env.example を .env にコピーし、テキストエディタ（メモ帳でOK）で開きます。

copy backend\.env.example backend\.env

3 APIキーを追加

少なくとも1つの翻訳プロバイダーが必要です。Gemini（無料）の設定方法：

  1. aistudio.google.com/apikey にアクセス
  2. Googleアカウントでログイン
  3. 「APIキーを作成」をクリック
  4. キーをコピーして .env に貼り付け：
# backend/.env
GEMINI_API_KEY=AIzaSy...

ファイルを保存して完了です！

4 アプリを再起動

アプリを閉じて、もう一度 jpeigo-slides.exe を実行してください。右上のヘルスインジケーターが 緑色 になれば、キーが認識されています。
